# 로그 레벨 설정

SDK의 동작을 확인하기 위해 아래와 같이 로그 레벨을 설정할 수 있습니다. *setEnv*를 통해 설정되는 기본 로그 레벨을 사용하는 것을 권장하지만, 필요한 경우 로그 레벨을 커스터마이징할 수 있습니다.

| **로그 레벨** | **설명** |
| ---| --- |
| `Verbose` | 모든 로그 항목을 출력합니다. |
| `Debug` | `Debug` 레벨 이상의 모든 로그 항목을 출력합니다. |
| `Info` | `Info` 레벨 이상의 모든 로그 항목을 출력합니다. |
| `Warn` | `Warn` 레벨 이상의 모든 로그 항목을 출력합니다. |
| `Error` | `Error` 레벨의 로그 항목만 출력합니다. |
| `Off` | 어떤 로그 항목도 출력하지 않습니다. |

로그 레벨은 설정 직후에 발생하는 로그부터 즉시 적용됩니다. 예를 들어 로그 레벨을 *Verbose*로 설정하려면 아래 코드를 사용합니다:

```kotlin
FlowerSdk.setLogLevel("Verbose")
```
