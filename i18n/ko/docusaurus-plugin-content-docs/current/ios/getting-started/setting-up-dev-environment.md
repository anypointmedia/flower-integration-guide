# 개발 환경 설정

## FlowerSdk 종속성 설정

FlowerSdk는 GitHub을 통해 Swift Package Manager 형식으로 제공됩니다. 이를 적용하기 위해 아래와 같은 순서로 진행하면 됩니다.
1. **패키지 종속성 추가:** 상단 메뉴에서 **File** > **Add Package Dependencies...** 를 선택하고 Flower SDK 저장소 URL을 입력합니다.
[https://github.com/anypointmedia/flower-sdk-ios](https://github.com/anypointmedia/flower-sdk-ios)![](/img/docs/7a0e14ea-f70f-4b19-95cd-0e20d65e9979.png)
2. **프로젝트에 Flower SDK 추가:** 다음 화면에서 FlowerSdk를 선택하고 iOS 앱 프로젝트에 추가합니다.
![](/img/docs/c10ff499-4b92-48b1-9809-ce595a4c533c.png)
3. **프로젝트 설정에서 Flower SDK 확인:** iOS 앱 프로젝트의 **General** 설정 탭에서 FlowerSdk가 추가되었는지 확인합니다.
![](/img/docs/7cb84cf4-e95c-451b-b139-85ae8cea757f.png)
