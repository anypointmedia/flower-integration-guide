# 콘텐츠 유형과 사용 예시

FLOWER가 지원하는 콘텐츠 유형별 광고 서비스와 실제 활용 시나리오를 소개합니다.

## 실시간 채널

### 본 스트림 대체 광고

광고 마커(예: SCTE-35)를 활용하여 본 콘텐츠 스트림 내 기존 광고 구간을 대체하는 방식입니다. 두 전달 방식 모두 지원하며, 처리 메커니즘이 다릅니다.

*   **Unicast (HLS/DASH)**: SDK가 재생목록 매니페스트(m3u8 또는 mpd)를 가공하여 광고 마커를 감지하고 스트림 내 광고 구간을 대체합니다.
*   **Satellite/IPTV**: SDK가 SCTE-35, DTMF 큐톤, EPG 기반 가상 큐를 감지하고 스트림 내 광고 구간을 대체합니다.

#### 사용 예시

*   지상파·케이블 채널의 기존 광고 구간을 자사 광고 또는 타겟팅 광고로 대체하여 추가 수익을 확보합니다.
*   FAST 채널에서 콘텐츠 사이에 프로그래매틱 광고를 삽입하여 광고 인벤토리를 극대화합니다.
*   라이브 스포츠 중계의 하프타임·쿼터 브레이크 등 자연스러운 휴식 구간에 프리미엄 광고를 배치합니다.
*   지역별로 다른 광고를 송출하여 로컬 광고주의 수요를 충족합니다.

### 채널 진입 광고

시청자가 실시간 채널에 진입하는 순간, 본 콘텐츠가 재생되기 전에 노출되는 광고입니다. 광고가 끝나면 본 스트림으로 자연스럽게 전환됩니다. Unicast (HLS/DASH) 전달 방식에서 사용할 수 있습니다.

#### 사용 예시

*   무료 채널에 진입 광고를 설정하여 채널 전환 시점마다 광고 수익을 창출합니다.
*   광고 기반 요금제(AVOD)를 운영하는 OTT 서비스에서 채널 진입 시점을 프리미엄 인벤토리로 활용합니다.
*   신규 오리지널 콘텐츠나 캠페인 프로모션을 채널 진입 시점에 노출하여 시청자 인지도를 높입니다.

### 아날로그 큐 및 푸시 기반 광고

Satellite/IPTV(위성 DTH 및 IPTV 멀티캐스트) 전달 방식에 특화된 광고 유형으로, 백엔드 시스템이 클라이언트 디바이스의 광고 재생을 직접 트리거합니다.

*   **디지털 큐(SCTE-35)**: TS 스트림에 포함된 SCTE-35 패킷을 기반으로 광고 구간을 대체합니다.
*   **아날로그 큐**: H/E(Head-End) 시스템에서 감지된 광고 큐가 Anypoint 플랫폼을 통해 디바이스로 전달되어 광고 구간을 대체합니다.
*   **푸시 기반 (가상 큐)**: 백엔드가 방송 프로그램 편성표(EPG)를 기반으로 Anypoint 플랫폼을 통해 디바이스로 전달되어 광고 구간을 대체합니다.

#### 사용 예시

*   H/E 시스템이 감지한 방송 광고 구간에 맞춰 셋톱박스에서 타겟팅 광고를 재생합니다.
*   편성표 기반으로 광고 브레이크를 예약하여, 스트림 내 마커가 없는 채널에서도 정확한 시점에 광고를 송출합니다.
*   프라임타임 시간대에 집중적으로 광고를 배치하여 CPM을 극대화합니다.

## VOD

VOD 콘텐츠의 재생 전·중·후 시점에 광고를 삽입할 수 있습니다.

*   **Pre-roll 광고**: 본 콘텐츠 시작 전에 재생되는 광고입니다.
*   **Mid-roll 광고**: 콘텐츠 내 일정 간격 또는 지정된 삽입 지점에 재생되는 광고입니다.
*   **Post-roll 광고**: 본 콘텐츠 종료 후에 재생되는 광고입니다.

#### 사용 예시

*   장편 영화나 드라마 시리즈에 pre-roll·mid-roll·post-roll 광고를 조합하여 에피소드당 수익을 극대화합니다.
*   자연스러운 장면 전환 지점(챕터 구간)에 mid-roll 광고를 배치하여 시청 이탈을 최소화합니다.
*   AVOD 서비스에서 무료 콘텐츠에 광고를 삽입하고, 프리미엄 구독자에게는 광고 없는 경험을 제공합니다.
*   숏폼 콘텐츠에는 pre-roll만 적용하여 간결한 광고 경험을 유지합니다.

## 웹페이지 / 앱

웹페이지와 앱 내에 비디오 광고를 삽입할 수 있습니다.

#### 사용 예시

*   포털 사이트 메인 페이지에 인라인 비디오 광고를 배치하여 높은 트래픽을 광고 수익으로 전환합니다.
*   뉴스·블로그 기사 본문 중간에 비디오 광고를 삽입하여 콘텐츠 소비 흐름 안에서 자연스럽게 노출합니다.
*   게임 앱에서 레벨 전환·보상 지급 시점에 전면(인터스티셜) 광고를 표시하여 유저 참여도를 유지하면서 수익을 확보합니다.
*   앱 실행 시 마스트헤드 영역에 브랜드 비디오 광고를 노출하여 프리미엄 광고 인벤토리를 제공합니다.
*   무료 앱에서 특정 기능 이용 전 광고 시청을 요구하는 리워드형 광고를 적용합니다.
